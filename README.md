# Запуск проекта

```sh
cd project folder
npm install
```

```sh
cd project folder
docker-compose -f docker-compose.databases.yml up
database.env - конфиг для локального подключения к postgres
```

```sh
cd project folder
npm run dev
```


```sh
browser localhost:{{PORT}}/graphql - оболочка запросов
```


### Тестирование

```sh
npm run test
```
